## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/ziepziep/morsebit-beta** and import

## Edit this project ![Build status badge](https://github.com/ziepziep/morsebit-beta/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/ziepziep/morsebit-beta** and click import

## Använd som tillägg

Denna databas kan läggas till som ett **tillägg** i MakeCode.

* öppna [https://makecode.microbit.org/](https://makecode.microbit.org/)
* klicka på **Nytt projekt**
* klicka på **Tillägg** under kugghjulsmenyn
* sök efter **https://github.com/viktorbakken/morsebit-beta** och importera

## Redigera detta projekt ![Byggets statusikon](https://github.com/viktorbakken/morsebit-beta/workflows/MakeCode/badge.svg)

Redigera den här databasen i MakeCode.

* öppna [https://makecode.microbit.org/](https://makecode.microbit.org/)

* klistra in **https://github.com/viktorbakken/morsebit-beta** och klicka på importera
